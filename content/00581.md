このBuilderで対応している言語は

* Java
* NodeJS
* Golang
* .NET Core

またStackとしては`gcr.io/paketo-buildpacks/run:base-cnb`が使用されます。

BuilderとStackを予めpullしておきます。

```
docker pull gcr.io/paketo-buildpacks/builder:base
docker pull gcr.io/paketo-buildpacks/run:base-cnb
```
