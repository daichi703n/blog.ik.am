memo

### JSON -> YAML

```bash
cat foo.json | ruby -ryaml -rjson -e 'puts YAML.dump(JSON.load(ARGF))'
```

### YAML -> JSON

```bash
cat foo.yml | ruby -ryaml -rjson -e 'puts JSON.dump(YAML.load(ARGF))'
```

pretty printする場合は

```bash
cat foo.yml | ruby -ryaml -rjson -e 'puts JSON.pretty_generate(YAML.load(ARGF))'
```
