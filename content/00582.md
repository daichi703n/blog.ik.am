このBuilderで対応している言語は

* Java
* NodeJS
* Golang
* .NET Core
* PHP
* Staticfile (httpd, nginx)

またStackとしては`gcr.io/paketo-buildpacks/run:full-cnb-cf`が使用されます。

BuilderとStackを

```
docker pull gcr.io/paketo-buildpacks/builder:full-cf
docker pull gcr.io/paketo-buildpacks/run:full-cnb-cf
```
